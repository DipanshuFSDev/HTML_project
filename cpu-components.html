CPU COMPONENTS
Sr.no Name Images Description
1 CPU a computer's main processor that performs instructions from software and hardware programs.
2 Motherboard the backbone that ties the computer's components together at one spot and allows them to talk to each other.
3 CPU Cooler heat sinks which are specifically designed to remove waste heat generated by a central processing unit (CPU).
4 Power Supply Unit A "power supply unit" (PSU) is a crucial component in electronic devices that converts the incoming alternating current (AC) electricity from a wall outlet into the regulated direct current (DC) voltage needed to power various internal components.
5 Video Card A video card, also known as a graphics card or graphics processing unit (GPU), is an essential component of a computer that enables the user to view visual information on the monitor. It processes data in the form of pixels, which make up images.
6 Computer Case a box that contains and protects the internal components of a computer. It also provides a framework to cool the components.
7 Sound Card a computer component responsible for generating and recording audio. It enables users to connect analog speakers, headphones and microphones to their computer.
8 GPU a hardware component that processes and displays images, videos, and animations on a computer. GPUs are also known as video cards.
9 Chipset a collection of electronic components on a motherboard that manages the flow of data between the CPU, GPU, RAM, storage, and peripherals.
10 RAM a computer's short-term memory that stores data for the CPU to access quickly.
11 Optical Drive a piece of computer hardware that reads and writes data on optical discs. The most common use of an optical drive is to install software or operating systems from a disc. Optical discs are also used to store music, movies, and photos.
12 Hard Disk Drive a type of data storage device that is used in laptops and desktop computers. An HDD is a “non-volatile” storage drive, which means it can retain the stored data even when no power is supplied to the device.
13 M.2 Solt a small, rectangular expansion card that connects solid-state drives (SSDs) and other devices to a computer's motherboard.
14 RAID Controller a hardware device or software program that manages hard drives or solid-state drives in a computer or storage array
15 CMOS Battery a small, coin-shaped battery that stores important system information on a computer's motherboard.
